⚙️ Installation Guide

                    
                    
Step 1: Locate CS2 Config Directory

                    

                        
Navigate to CS2 configuration folder:

                        
Path: Steam/steamapps/common/Counter-Strike Global Offensive/game/csgo/cfg/

                        
Access via: Steam → CS2 → Properties → Browse Local Files → game/csgo/cfg

                    


                    
Step 2: Extract Configuration

                    

                        
1. Download the autoexec.zip file from GitHub

                        
2. Extract all contents to the cfg directory

                        
3. Ensure autoexec.cfg is directly in the cfg folder

                        
4. Verify movement/ directory is also present

                    


                    
Step 3: Enable Autoexec

                    

                        
Method 1 - Launch Options (Recommended):

                        
Steam → CS2 → Properties → Launch Options → Add: +exec autoexec

                    

                    

                        
Method 2 - Manual Console:

                        
In-game console (~) → exec autoexec

                    


                    

                        ⚠️ Important: Some bindings require sv_cheats 1 enabled on the server.
